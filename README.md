


# 🚀 AI Tool Hub

A modern and responsive digital tools marketplace built with **React, Tailwind CSS, and DaisyUI**.
Users can browse premium digital products, add them to cart, remove items, and proceed to checkout with a smooth UI experience.

---

## 📖 Project Description

AI Tool Hub is a clean and user-friendly web application where users can explore premium digital products like resume builders, logo makers, video editors, and more.

The project includes:

* Product cards layout
* Cart functionality
* Dynamic cart count in navbar
* Product and cart toggle section
* Toast notifications
* Checkout system

---

## 🛠️ Technologies Used

* **React.js**
* **Tailwind CSS**
* **DaisyUI**
* **React Icons**
* **React Toastify**
* **Vite**

---

## ✨ Key Features

### 1. 🛒 Smart Cart System

Users can add products to cart, remove selected items, and checkout instantly.

### 2. 🔄 Product & Cart Toggle

Toggle between **Products Section** and **Cart Section** with dynamic UI updates.

### 3. 🔔 Real-time Alerts

Integrated **React Toastify** for add, remove, and checkout notifications.

---

## 📸 UI Highlights

* Responsive Navbar
* Cart count badge
* 3-column product cards
* Empty cart message with icon
* Modern step section
* Smooth user experience
